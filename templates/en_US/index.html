Welcome to CatDustry!

CatDustry is a unique network infrastructure that integrates various gaming and communication services in one space. It encompasses several powerful servers and solutions, providing users with access to a variety of features and capabilities for communication and cooperative play.

  • Mindustry Servers: CatDustry hosts Mindustry servers where users can build production chains, battle, and interact. Players develop bases, automate mining, and create resources.

  • Telegram Integration: Telegram integration lets users manage gameplay through the messenger, simplifying communication and server interactions with commands and notifications.

  • Factorio and Satisfactory Servers: CatDustry includes servers for Factorio and Satisfactory, enabling players to collaborate on building and optimizing manufacturing processes.

  • File Sharing: CatDustry offers a secure file-sharing platform for exchanging game mods, schematics, and other materials conveniently.

  • Matrix Server: The Matrix server supports chat, calls, and integrations, offering a platform for communication, gaming discussions, and finding like-minded users.

CatDustry is a space where games, communication and file sharing are connected in a single ecosystem, allowing users to get the maximum pleasure from their hobbies.
